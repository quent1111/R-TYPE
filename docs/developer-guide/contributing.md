# Contributing to R-TYPE

Thank you for your interest in contributing to R-TYPE!

## Getting Started

1. Fork the repository
2. Clone your fork
3. Install git hooks: `./scripts/install-hooks.sh`
4. Create a feature branch
5. Make your changes
6. Submit a pull request

## Code Style

We follow the **Google C++ Style Guide**. All code is automatically checked by:
- clang-format (formatting)
- clang-tidy (static analysis)
- Pre-commit hooks

See [Code Style Guide](code-style.md) for details.

## Pull Request Process

1. Ensure your code passes all checks
2. Update documentation if needed
3. Add tests for new features
4. Request review from maintainers

## Need Help?

- Check existing issues
- Join our Discord
- Ask questions in discussions
