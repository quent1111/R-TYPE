# Scripts Reference

Quick reference for all utility scripts in the R-TYPE project.

For detailed documentation, see [scripts.md](https://github.com/quent1111/R-TYPE/blob/main/scripts/scripts.md) in the repository.

## Quick Commands

### Documentation

```bash
# Install dependencies
./scripts/docs.sh install

# Start dev server
./scripts/docs.sh serve

# Build docs
./scripts/docs.sh build
```

### Code Quality

```bash
# Format code
./scripts/format.sh -f

# Run linter
./scripts/format.sh -t

# Format + lint
./scripts/format.sh -a
```

### Setup

```bash
# Install git hooks
./scripts/install-hooks.sh
```
